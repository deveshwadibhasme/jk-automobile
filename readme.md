# Vehicle Management & Payment Platform (Full Stack) (Pay Before Download)

This repository contains a **production-style full-stack application** consisting of:

- **Admin Dashboard** – manage vehicles, files, and system data
- **Client Application** – user-facing app with authentication and payments
- **Backend API** – handles auth, database, uploads, and payment processing
- **User** Can Browse Vehicles and Buy Bin files by paying respective fare

---

## Tech Stack

### Frontend (Admin & Client)
- React (Vite)
- React Router
- Context API (Authentication)
- Tailwind CSS
- Axios

### Backend
- Node.js (ESM)
- Express
- MySQL
- JWT Authentication
- Multer (file uploads)
- ImageKit + AWS S3
- Razorpay (payments)
- Nodemailer / Resend
